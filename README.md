# securityplus-network-Cibersecurity-labs
Hands-on cybersecurity labs based on CompTIA Security+ (Network Security Fundamentals). This repository organizes practical exercises, explanations, and step-by-step implementations to teach core network security concepts, including protocols, devices, firewalls, attacks, and best practices.

## References

- Ciampa, Mark. *CompTIA Security+ Study Guide*, 5th Edition.  
- SANS Institute. Security Policy Templates. https://www.sans.org/security-resources/policies/
